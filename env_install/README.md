# This folder shows how to install a neovim based golang ide

```
sh neovim_go.sh
```

# [参考网址](https://github.com/plentiform/go-ide)

neovim 的基本用法：

`nt`：打开文件浏览器

`ctrl-p`：打开文件（这个会比较方便）

`/`：字符串匹配

`gd`：go to definition，查看某个值的定义；

`ctrl-o`：用于返回上面的 `gd`

`tab, shift tab`：用于选择提示

`ctrl-j, ctrl-k`：代码片段快捷键，比如敲了 if，他自动补全
